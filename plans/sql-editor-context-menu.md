# SQL 编辑器自定义右键菜单设计方案

## 背景

## 菜单结构设计

```
┌─────────────────────────┐

│ 剪切            Ctrl+X   │
│ 复制            Ctrl+C   │
│ 粘贴            Ctrl+V   │

```

## 实现方案

### 技术方案

使用 Ant Design Vue 的 `a-dropdown` + `a-menu` 组件，通过监听 Monaco Editor 容器的 `contextmenu` 事件触发。

### 修改文件

- `src/components/editor/SqlEditor.vue` — 添加自定义右键菜单

### 实现步骤

1. **添加右键菜单状态变量**：`contextMenuVisible`、`contextMenuX`、`contextMenuY`、`hasSelection`

2. **添加右键菜单模板**：在编辑器容器上监听 `@contextmenu` 事件，使用 `a-dropdown` 或自定义浮层显示菜单

3. **实现菜单功能**：
   - **剪切/复制/粘贴/全选**：调用 Monaco Editor 的 API（`editor.trigger` 或 document.execCommand）
   - **注释/取消注释**：对选中行添加/移除 `--` 前缀
   - **转大写/小写**：对选中文本进行大小写转换
   - **执行选中语句**：获取选中文本执行
   - 其余功能复用已有方法

4. **条件显示**：
   - 「执行选中语句」仅在有选中文本时显示
   - 「剪切」和「复制」仅在有选中文本时可用
   - 「粘贴」根据剪贴板状态决定是否可用
